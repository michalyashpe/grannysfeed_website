• 1) Get a laptop for your Granny

    Any simple laptop with the standard wifi connection and built-in camera & microphone should do the job. I got my granny a 100$ second-hand Asus Eee PC mini-laptop and it’s working great.

    Optional - get a high quality webcam & microphone to provide granny an enhanced experience. Operating System - this guide is intended for Windows XP, other OSs should work too with mild adjustments.

  • 2) Connect granny’s laptop to the internet

    In case you don’t have an internet connection at your granny’s house - ask your neighbours for their wifi password. I’m sure they’ll be happy to have your granny as a guest on their line, her mild usage of the internet should surely go unnoticeable ;-)

  • 3) Create a family Dropbox folder

    1. Download & install Dropbox client from https://www.dropbox.com/install
    2. Create a new account for granny & sign in
    3. Create a new folder and name it “Granny’s Feed Saver Pics”
    4. Share this folder with all family members (including you!)
    5. Put there some pics to start with. From now on everytime a family member takes a photo with a smartphone they can easily share it via the dropbox mobile app in seconds sending it directly to Granny’s screensaver!

  • 4) Set the screen saver

    1. Open Display in Control Panel.
    2. Choose the Screen Saver tab and click My Pictures Slideshow in the list.
    3. Click Settings to specify the newly created Dropbox folder - “Granny’s Feed Pics”
    4. Inside the wait tab set the time to 1 minute.

  • 5) Disable sleep mode

    We want the screen saver to always keep the family’s slide show on while idle, so sleep mode is not an option.

    1. Go to Start, Control Panel and click on Power Options
    2. On the Power Schemes tab, select Home/Office Desk in the Power Schemes drop-down box
    3. In the box labeled System standby, select Never
    4. Click OK and save your changes.

  • 6) Enable incoming calls

    1. Install Skype from http://www.skype.com/en/download-skype/skype-for-computer/
    2. Create a new user account for granny
    3. Add ranny’s contact list all family members and friends
    4. Allow contacts only to make calls and send messages (full guide here)
    5. Set auto answer calls with video (full guide here)

    This step would enable anyone on Granny’s contacts list to immediately initiate a call with granny and immediately see her. This could jeopardize granny’s privacy. Take this into consideration when applying this setting. In our case Granny’s laptop is located right in front of her sofa, where she sits in front of the TV and is really glad to be surprised with incoming video calls from us, her family

  • 7) Let Granny easily initiate video calls to the family

    Configure keyboard shortcut keys to call specific contacts:

    1. Download AutoHotKey at http://www.autohotkey.com
    2. Create a new file called “skype_script.ahk”
    3. For each contact granny wants to call add a new line like this:
      KEY::Run “C:\Program Files\Skype\Phone\Skype.exe” “/callto:SKYPE_USER&video=true”
      where KEY is the desired keyboard key in order to call a specific person (e.g F1. see list of all keys here) and SKYPE_USER is this person’s skype user name.
    4. Double-click the file to launch it. A new icon appears in the taskbar notification area, and the script is waiting to be triggered by keyboard clicks. Try it!

  • 8) Remote Access

    Murphy law says - If anything can go wrong it will.. I found it very helpful and extremely easy to access granny’s laptop when I’m away using Chrome Remote Desktop tool. Download and follow installation guide here.
